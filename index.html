VPC • EC2 • RDS • Route 53 • Client VPN

Project Overview

This website is deployed on an EC2 instance inside a public subnet. The backend database is hosted on Amazon RDS in a private subnet and secured through AWS Client VPN.

AWS Architecture

  • Custom VPC
  • Public & Private Subnets
  • EC2 Web Server
  • Amazon RDS (MySQL/PostgreSQL)
  • AWS Client VPN
  • Route 53 Domain & Subdomain

Services Used

Amazon EC2
Amazon RDS
Amazon VPC
Route 53
